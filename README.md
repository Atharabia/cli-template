# cli-template

A [pyportion](https://github.com/Atharabia/pyportion) template for scaffolding Python CLI projects.

## Structure

```
cli/
├── __main__.py        # Entry point
├── cli.py             # App setup
├── base/              # CommandBase & HandlerBase
├── commands/          # Commands
├── handler/           # Register handlers
├── core/              # Terminal output wrapper
└── models/            # App config
```
